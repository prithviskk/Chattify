To run the backend :(in developers mode)
npm run dev

To run the frontend:(in developers mode)
npm run dev 

To build the app:
npm run build

To start :
npm start

Setup the env variable :
MONGODB_URI=...
PORT=5001
JWT_SECRET=...

CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...

NODE_ENV=development
